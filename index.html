Project Description:

Data can be exported from Apple's Health app in XML format. The objective of this project is to create a website which the user can use to upload their data and visualize it. The website will also analyze the data to display fitness score and give personalized recommendations.

Why is our project important?

This website helps users track their daily activies. For example, you can easily information such as how many miles you walked, how many hours you slept, or how many calories you burned today. You can also see your minimum, maximum, and average heart rate.
